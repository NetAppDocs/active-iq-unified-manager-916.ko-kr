= Unified Manager 업그레이드 파일 다운로드
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager를 업그레이드하기 전에 NetApp 지원 사이트에서 Unified Manager 업그레이드 파일을 다운로드하세요.

.시작하기 전에
NetApp 지원 사이트에 대한 로그인 자격 증명이 있어야 합니다.

.단계
. NetApp 지원 사이트에 로그인하세요.
+
https://mysupport.netapp.com/site/products/all/details/activeiq-unified-manager/downloads-tab["NetApp 지원 사이트"]

. VMware vSphere에서 Unified Manager를 업그레이드하려면 다운로드 페이지로 이동하세요.
. 다운로드 `.iso` 업그레이드할 이미지를 만들고 vSphere Client에서 액세스할 수 있는 로컬 디렉토리나 네트워크 디렉토리에 저장합니다.
. 소프트웨어가 올바르게 다운로드되었는지 확인하려면 체크섬을 확인하세요.
