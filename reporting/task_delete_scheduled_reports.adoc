= 예약된 보고서 삭제
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
보고서가 예약된 후 보고서 일정 페이지에서 보고서를 삭제할 수 있습니다.

.시작하기 전에
* 애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.


.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *보고서 일정*을 클릭합니다.
+
image::../media/scheduled_reports_2.gif[예약된 보고서를 보여주는 UI 스크린샷입니다.]

+
[NOTE]
====
적절한 권한이 있다면 시스템에서 모든 보고서와 일정을 제거할 수 있습니다.

====
. 더 많은 아이콘을 클릭하세요image:../media/more_icon.gif[""] 제거하려는 일정에 대해.
. *삭제*를 클릭하세요.
. 결정을 확인하세요.
+
예약된 보고서는 목록에서 제거되고 더 이상 설정된 일정에 따라 생성 및 배포되지 않습니다.

+
[NOTE]
====
인벤토리 페이지에서 사용자 지정 보기를 삭제하면 해당 보기를 사용하는 사용자 지정 Excel 파일이나 예약된 보고서도 삭제됩니다.

====
