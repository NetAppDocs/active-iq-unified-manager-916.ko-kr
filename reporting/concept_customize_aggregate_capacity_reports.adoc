= 집계 용량 보고서 사용자 정의
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이러한 샘플 사용자 정의 보고서는 집계 스토리지 용량과 관련된 잠재적 문제를 식별하고 대응하는 데 도움이 됩니다.

이 섹션의 보고서는 스토리지 시스템 리소스를 모니터링하는 데 도움이 되는 집계 용량에 대한 보고서를 만드는 방법을 이해하는 데 도움이 되는 샘플일 뿐입니다.
