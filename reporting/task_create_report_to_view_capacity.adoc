= 집계 용량 표와 차트를 보여주는 보고서 만들기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
합계와 클러스터형 막대형 차트 형식을 사용하여 Excel 파일에서 용량을 분석하는 보고서를 만들 수 있습니다.

.시작하기 전에
* 애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.


다음 단계에 따라 Health: All Aggregates 보기를 열고, Excel에서 보기를 다운로드하고, 사용 가능한 용량 차트를 만들고, 사용자 지정 Excel 파일을 업로드하고, 최종 보고서를 예약합니다.

.단계
. 왼쪽 탐색 창에서 *저장소* > *집계*를 클릭합니다.
. *보고서* > *Excel 다운로드*를 선택합니다.
+
image::../media/download_excel_menu.png[보고서에서 Excel을 다운로드하는 방법을 보여주는 UI 스크린샷입니다.]

+
브라우저에 따라 파일을 저장하려면 *확인*을 클릭해야 할 수도 있습니다.

. 필요한 경우 *편집 활성화*를 클릭하세요.
. Excel에서 다운로드한 파일을 엽니다.
. 새 시트 만들기 (image:../media/excel_new_sheet_icon.png[""] ) 후에 `data` 시트를 만들고 *총 데이터 용량*이라고 이름을 지정합니다.
. 새로운 총 데이터 용량 시트에 다음 열을 추가합니다.
+
.. 총 데이터 용량(GB)
.. 약정 용량(GB)
.. 사용된 데이터 용량(GB)
.. 사용 가능한 데이터 용량(GB)


. 각 열의 첫 번째 행에 다음 수식을 입력하고, 데이터 시트(데이터!)를 참조하는지 확인하고, 캡처된 데이터에 대한 올바른 열 및 행 지정자를 참조하는지 확인합니다(총 데이터 용량은 열 E, 행 2~20에서 데이터를 가져옵니다).
+
.. =SUM(데이터!E$2:데이터!E$20)
.. =SUM(데이터!F$2:데이터!F$50)
.. =SUM(데이터!G$2:데이터!G$50)
.. =SUM(데이터!H$2:데이터!H$50)


+
이 수식은 현재 데이터를 기준으로 각 열의 합계를 계산합니다.



image::../media/capacitysums.png[현재 데이터를 기반으로 총 용량을 보여주는 데이터 시트의 UI 스크린샷입니다.]

. 데이터 시트에서 *총 데이터 용량(GB)* 및 *약정 용량(GB)* 열을 선택합니다.
. *삽입* 메뉴에서 *추천 차트*를 선택하고 *묶음 막대형 차트*를 선택합니다.
. 차트를 마우스 오른쪽 버튼으로 클릭하고 *차트 이동*을 선택하여 차트를 이동합니다. `Total Data Capacity` 시트.
. 차트를 선택하면 제공되는 *디자인* 및 *서식* 메뉴를 사용하여 차트의 모양을 사용자 지정할 수 있습니다.
. 만족스러우면 변경 사항을 적용하여 파일을 저장합니다.  파일 이름이나 위치를 변경하지 마세요.
+
image::../media/cluster_column_chart_2.png[총 데이터와 약정 용량을 보여주는 차트의 UI 스크린샷입니다.]

. Unified Manager에서 *보고서* > *Excel 업로드*를 선택합니다.
+
[NOTE]
====
Excel 파일을 다운로드한 것과 같은 보기에 있는지 확인하세요.

====
. 수정한 Excel 파일을 선택하세요.
. *열기*를 클릭하세요.
. *제출*을 클릭하세요.
+
*보고서* > *Excel 업로드* 메뉴 항목 옆에 체크 표시가 나타납니다.

+
image::../media/upload_excel.png[보고서에 Excel을 업로드하는 방법을 보여주는 UI 스크린샷입니다.]

. *예약된 보고서*를 클릭합니다.
. *일정 추가*를 클릭하면 보고서 일정 페이지에 새 행이 추가되어 새 보고서에 대한 일정 특성을 정의할 수 있습니다.
+
[NOTE]
====
보고서에 *XLSX* 형식을 선택하세요.

====
. 보고서 일정에 대한 이름을 입력하고 다른 보고서 필드를 완료한 다음 확인 표시(image:../media/blue_check.gif[""] ) 행의 끝에.
+
보고서는 테스트로 즉시 전송됩니다.  그 후, 보고서가 생성되어 지정된 빈도로 등록된 수신자에게 이메일로 전송됩니다.



보고서에 표시된 결과를 바탕으로 네트워크 전반에서 사용 가능한 용량을 가장 효과적으로 활용하는 방법을 알아보고 싶을 수도 있습니다.
