= 스토리지 VM 보고서 사용자 정의
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
볼륨 정보를 분석하고 전반적인 상태와 스토리지 가용성을 확인하기 위해 스토리지 VM 보고서를 만들 수 있습니다.  예를 들어, 최대 볼륨 수에 도달한 SVM을 표시하고 중지된 SVM을 분석하는 보고서를 만들 수 있습니다.
