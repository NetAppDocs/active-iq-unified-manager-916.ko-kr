= 보고서 일정 관리
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
보고서 일정 페이지에서 보고서 일정을 관리할 수 있습니다.  기존 일정을 보고, 수정하고, 삭제할 수 있습니다.

[NOTE]
.시작하기 전에
====
보고서 일정 페이지에서는 새 보고서를 예약할 수 없습니다.  예약된 보고서는 객체 인벤토리 페이지에서만 추가할 수 있습니다.

====
* 애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.


.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *보고서 일정*을 클릭합니다.
. 보고서 일정 페이지에서:
+
[cols="2*"]
|===
| 만약 당신이 원한다면... | 그 다음에... 


 a| 
기존 일정 보기
 a| 
스크롤 막대와 페이지 컨트롤을 사용하여 기존 보고서 목록을 스크롤합니다.



 a| 
기존 일정 편집
 a| 
.. 더 많은 아이콘을 클릭하세요image:../media/more_icon.gif[""] 사용하려는 일정에 맞게.
.. *편집*을 클릭하세요.
.. 필요한 변경 사항을 적용합니다.
.. 변경 사항을 저장하려면 확인 표시를 클릭하세요.




 a| 
기존 일정 삭제
 a| 
.. 더 많은 아이콘을 클릭하세요image:../media/more_icon.gif[""] 사용하려는 일정에 맞게.
.. *삭제*를 클릭하세요.
.. 결정을 확인하세요.


|===
