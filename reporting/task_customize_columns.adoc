= 열 사용자 정의
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
보고서에 사용할 열을 선택하려면 *표시/숨기기*를 사용하세요.  인벤토리 페이지에서 열을 끌어서 다시 정렬하세요.

.단계
. 열을 추가하거나 제거하려면 *표시/숨기기*를 클릭하세요.
+
image::../media/show_hide_3.png[열을 추가하거나 제거하는 방법을 보여주는 UI 스크린샷입니다.]

. 인벤토리 페이지에서 열을 끌어서 보고서에서 원하는 순서대로 다시 정렬합니다.
. 변경 사항을 저장하려면 저장되지 않은 보기의 이름을 지정하세요.
