= 예약된 보고서 검색
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이름, 보기 이름, 개체 유형 또는 수신자를 기준으로 예약된 보고서를 검색할 수 있습니다.

. 왼쪽 탐색 창에서 *저장소 관리* > *보고서 일정*을 클릭합니다.
. *예약된 보고서 검색* 텍스트 필드를 사용하세요.
+
[cols="2*"]
|===
| 보고서를 찾으려면... | 노력하다 ... 


 a| 
일정 이름
 a| 
보고서 일정 이름의 일부를 입력하세요.



 a| 
보기 이름
 a| 
보고서 보기 이름의 일부를 입력하세요.  기본 보기와 사용자 정의 보기가 보기 목록에 나타납니다.



 a| 
받는 사람
 a| 
이메일 주소의 일부를 입력하세요.



 a| 
파일 유형
 a| 
"PDF", "CSV" 또는 "XLSX"를 입력하세요.

|===
. 일정 이름이나 형식 등의 열 제목을 클릭하면 해당 열을 기준으로 보고서를 오름차순이나 내림차순으로 정렬할 수 있습니다.
