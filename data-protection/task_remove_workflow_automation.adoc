= Unified Manager에서 OnCommand Workflow Automation 설정 제거
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
더 이상 Workflow Automation을 사용하지 않으려면 Unified Manager에서 OnCommand Workflow Automation 설정을 제거할 수 있습니다.

.시작하기 전에
애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

.단계
. 왼쪽 탐색 창에서 왼쪽 설정 메뉴의 *일반* > *워크플로 자동화*를 클릭합니다.
. *워크플로 자동화* 페이지에서 *설치 제거*를 클릭합니다.
