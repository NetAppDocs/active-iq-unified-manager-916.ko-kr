= 작업 모니터링
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
작업 페이지를 사용하면 작업 상태를 모니터링하고 스토리지 서비스 유형, 상태, 제출 시간, 완료 시간과 같은 작업 속성을 보고 작업이 성공적으로 완료되었는지 확인할 수 있습니다.

.시작하기 전에
애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

.단계
. 왼쪽 탐색 창에서 *보호* > *작업*을 클릭합니다.
+
작업 페이지가 표시됩니다.

. 현재 실행 중인 작업의 상태를 확인하려면 *상태* 열을 확인하세요.
. 특정 직무에 대한 세부 정보를 보려면 직무 이름을 클릭하세요.
+
작업 세부 정보 페이지가 표시됩니다.
