= 지원되는 브라우저
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager 웹 UI에 액세스하려면 지원되는 브라우저를 사용하세요.

상호 운용성 매트릭스에는 지원되는 브라우저 버전 목록이 있습니다.

http://mysupport.netapp.com/matrix["mysupport.netapp.com/매트릭스"^]

모든 브라우저에서 팝업 차단 기능을 비활성화하면 소프트웨어 기능이 제대로 표시됩니다.

SAML 인증을 위해 Unified Manager를 구성하여 ID 공급자(IdP)가 사용자를 인증할 수 있도록 계획하는 경우 IdP에서 지원하는 브라우저 목록도 확인해야 합니다.
