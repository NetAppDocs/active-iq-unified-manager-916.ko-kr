= 주석 규칙 편집
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
주석 규칙을 편집하여 조건 그룹과 조건 그룹 내의 조건을 수정하고, 스토리지 객체에 주석을 추가하거나 스토리지 객체에서 주석을 제거할 수 있습니다.

.시작하기 전에
애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

연관된 주석 규칙을 편집하면 주석이 저장소 개체에서 분리됩니다.

.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *주석*을 클릭합니다.
. *주석 규칙* 탭에서 편집하려는 주석 규칙을 선택한 다음 *작업* > *편집*을 클릭합니다.
. *주석 규칙 편집* 대화 상자에서 필요에 따라 규칙 이름, 주석 이름 및 값, 조건 그룹, 조건을 변경합니다.
+
주석 규칙에 대한 대상 개체 유형을 변경할 수 없습니다.

. *저장*을 클릭하세요.
