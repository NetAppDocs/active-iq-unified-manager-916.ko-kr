= 기술 지원팀에 지원 번들을 보내세요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
문제에 AutoSupport 메시지에서 제공하는 것보다 더 자세한 진단 및 문제 해결 정보가 필요한 경우 기술 지원팀에 지원 번들을 보낼 수 있습니다.

.시작하기 전에
기술 지원팀에 문의하려면 지원 번들에 액세스할 수 있어야 합니다.

기술 지원 웹사이트를 통해 생성된 사례 번호가 있어야 합니다.

.단계
. NetApp 지원 사이트에 로그인하세요.
. 파일을 업로드하세요.
