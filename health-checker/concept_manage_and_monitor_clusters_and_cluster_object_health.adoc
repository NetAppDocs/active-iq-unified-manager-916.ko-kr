= 클러스터 및 클러스터 개체 상태 관리 및 모니터링
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager는 주기적 API 쿼리와 데이터 수집 엔진을 사용하여 클러스터에서 데이터를 수집합니다.  Unified Manager 데이터베이스에 클러스터를 추가하면 이러한 클러스터의 가용성 및 용량 위험을 모니터링하고 관리할 수 있습니다.
