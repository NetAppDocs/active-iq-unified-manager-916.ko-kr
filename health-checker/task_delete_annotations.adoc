= 주석 삭제
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
더 이상 필요하지 않은 사용자 정의 주석과 해당 값을 삭제할 수 있습니다.

.시작하기 전에
* 애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.
* 주석 값은 다른 주석이나 그룹 규칙에서 사용할 수 없습니다.


.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *주석*을 클릭합니다.
. *주석* 탭에서 삭제하려는 주석을 선택합니다.
+
선택한 주석의 세부 정보가 표시됩니다.

. *작업* > *삭제*를 클릭하여 선택한 주석과 해당 값을 삭제합니다.
. 경고 대화 상자에서 *예*를 클릭하여 삭제를 확인합니다.
