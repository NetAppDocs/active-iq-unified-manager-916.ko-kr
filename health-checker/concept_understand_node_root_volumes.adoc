= 노드 루트 볼륨 이해
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager를 사용하여 노드 루트 볼륨을 모니터링할 수 있습니다.  가장 좋은 방법은 노드 루트 볼륨에 노드가 다운되는 것을 방지할 만큼 충분한 용량이 있어야 한다는 것입니다.

노드 루트 볼륨의 사용된 용량이 전체 노드 루트 볼륨 용량의 80%를 초과하면 노드 루트 볼륨 공간이 거의 가득 찼습니다 이벤트가 생성됩니다.  이벤트에 대한 알림을 구성하여 알림을 받을 수 있습니다.  ONTAP 시스템 관리자나 ONTAP CLI를 사용하여 노드가 다운되는 것을 방지하기 위한 적절한 조치를 취할 수 있습니다.


NOTE: 클러스터가 ONTAP 9.14.1 버전 이상을 실행하는 경우 노드 루트 볼륨 모니터링 기능을 사용할 수 없습니다.
