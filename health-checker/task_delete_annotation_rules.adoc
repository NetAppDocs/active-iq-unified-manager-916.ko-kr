= 주석 규칙 삭제
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
규칙이 더 이상 필요하지 않으면 Active IQ Unified Manager 에서 주석 규칙을 삭제할 수 있습니다.

.시작하기 전에
애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

주석 규칙을 삭제하면 주석이 연결 해제되고 저장소 개체에서 제거됩니다.

.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *주석*을 클릭합니다.
. *주석 규칙* 탭에서 삭제하려는 주석 규칙을 선택한 다음 *삭제*를 클릭합니다.
. *경고* 대화 상자에서 *예*를 클릭하여 삭제를 확인합니다.
