= 개별 저장소 객체에 수동으로 주석 추가
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
주석 규칙을 사용하지 않고도 선택한 볼륨, 클러스터 및 SVM에 수동으로 주석을 달 수 있습니다.  단일 저장소 개체 또는 여러 저장소 개체에 주석을 달 수 있으며, 주석에 필요한 이름-값 쌍 조합을 지정할 수 있습니다.

.시작하기 전에
애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

.단계
. 주석을 달고 싶은 저장소 개체로 이동합니다.
+
[cols="2*"]
|===
| 주석을 추가하려면... | 이렇게 하세요... 


 a| 
클러스터
 a| 
.. *저장소* > *클러스터*를 클릭합니다.
.. 하나 이상의 클러스터를 선택하세요.




 a| 
볼륨
 a| 
.. *저장소* > *볼륨*을 클릭합니다.
.. 하나 이상의 볼륨을 선택하세요.




 a| 
SVM
 a| 
.. *저장소* > *SVM*을 클릭합니다.
.. 하나 이상의 SVM을 선택하세요.


|===
. *주석*을 클릭하고 이름-값 쌍을 선택합니다.
. *적용*을 클릭하세요.
