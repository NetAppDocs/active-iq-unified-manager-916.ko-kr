= 웹 UI 및 유지 관리 콘솔을 통해 지원 번들을 보내세요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
AutoSupport 메시지에서 제공하는 것보다 더 자세한 진단 및 문제 해결이 필요한 경우 지원 번들을 보내야 합니다.  Unified Manager 웹 UI와 유지 관리 콘솔을 사용하여 기술 지원팀에 지원 번들을 보낼 수 있습니다.

Unified Manager는 한 번에 최대 2개의 전체 지원 번들과 3개의 경량 지원 번들을 저장합니다.
