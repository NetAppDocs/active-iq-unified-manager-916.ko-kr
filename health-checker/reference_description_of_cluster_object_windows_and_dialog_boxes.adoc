= 클러스터 객체 창 및 대화 상자 설명
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
각 스토리지 객체 페이지에서 모든 클러스터와 클러스터 객체를 볼 수 있습니다.  해당 스토리지 객체 세부 정보 페이지에서 세부 정보를 볼 수도 있습니다.  이제 INVENTORY의 STORAGE 및 PROTECTION 섹션에서 시스템 관리자 사용자 인터페이스를 시작할 수 있습니다.

* 클러스터 인벤토리, 클러스터 상태 및 클러스터 성능 페이지
* 집계 재고, 집계 상태 및 집계 성과 페이지
* 볼륨 인벤토리, 볼륨 상태 및 볼륨 성능 페이지
* 노드 인벤토리 및 노드 성능 페이지
* StorageVM 인벤토리, StorageVM 상태 및 StorageVM 성능 페이지
* 보호 관계 페이지
