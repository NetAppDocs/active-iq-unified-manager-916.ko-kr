= SAML 인증에 사용되는 ID 공급자 변경
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager가 원격 사용자를 인증하는 데 사용하는 ID 공급자(IdP)를 변경할 수 있습니다.

.시작하기 전에
* IdP URL과 메타데이터가 있어야 합니다.
* IdP에 액세스할 수 있어야 합니다.


새로운 IdP는 Unified Manager를 구성하기 전이나 후에 구성할 수 있습니다.

.단계
. 왼쪽 탐색 창에서 *일반* > *SAML 인증*을 클릭합니다.
. Unified Manager 서버를 IdP에 연결하는 데 필요한 새로운 IdP URI와 IdP 메타데이터를 입력합니다.
+
Unified Manager 서버에서 IdP에 직접 액세스할 수 있는 경우 IdP URL을 입력한 후 *IdP 메타데이터 가져오기* 버튼을 클릭하면 IdP 메타데이터 필드가 자동으로 채워집니다.

. Unified Manager 메타데이터 URI를 복사하거나 메타데이터를 XML 텍스트 파일에 저장합니다.
. *구성 저장*을 클릭합니다.
+
구성을 변경할 것인지 확인하는 메시지 상자가 표시됩니다.

. *OK*를 클릭하세요.


새로운 IdP에 액세스하여 Unified Manager 서버 URI와 메타데이터를 입력하여 구성을 완료합니다.

다음에 권한이 있는 원격 사용자가 Unified Manager 그래픽 인터페이스에 액세스하려고 하면 이전 IdP 로그인 페이지 대신 새 IdP 로그인 페이지에 자격 증명을 입력하게 됩니다.
