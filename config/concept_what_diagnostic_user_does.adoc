= 진단 사용자 기능
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
진단 액세스의 목적은 기술 지원팀이 문제 해결을 도울 수 있도록 하는 것이며, 기술 지원팀의 지시가 있을 때에만 사용해야 합니다.

진단 사용자는 기술 지원팀의 지시에 따라 문제 해결을 위해 OS 수준 명령을 실행할 수 있습니다.
