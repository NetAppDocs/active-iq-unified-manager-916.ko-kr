= 사용자 액세스 관리
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Active IQ Unified Manager 에 대한 사용자 액세스를 제어하기 위해 역할을 만들고 기능을 할당할 수 있습니다.  Unified Manager 내에서 선택한 개체에 액세스하는 데 필요한 기능을 갖춘 사용자를 식별할 수 있습니다.  이러한 역할과 기능을 가진 사용자만 Unified Manager에서 개체를 관리할 수 있습니다.
