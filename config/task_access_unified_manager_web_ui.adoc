= Unified Manager 웹 UI에 액세스하세요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager를 설치한 후 웹 UI에 액세스하여 Unified Manager를 설정하고 ONTAP 시스템 모니터링을 시작할 수 있습니다.

.시작하기 전에
* 웹 UI에 처음 접속하는 경우 유지 관리 사용자(Linux 설치의 경우 umadmin 사용자)로 로그인해야 합니다.
* 정규화된 도메인 이름(FQDN)이나 IP 주소 대신 짧은 이름을 사용하여 사용자가 Unified Manager에 액세스하도록 허용하려는 경우 네트워크 구성에서 이 짧은 이름을 유효한 FQDN으로 확인해야 합니다.
* 서버가 자체 서명된 디지털 인증서를 사용하는 경우 브라우저에 해당 인증서를 신뢰할 수 없다는 경고가 표시될 수 있습니다.  위험을 인정하고 계속 접근하거나, 서버 인증을 위해 인증 기관(CA) 서명 디지털 인증서를 설치할 수 있습니다.


.단계
. 설치가 끝나면 표시되는 URL을 사용하여 브라우저에서 Unified Manager 웹 UI를 시작합니다.  URL은 Unified Manager 서버의 IP 주소 또는 정규화된 도메인 이름(FQDN)입니다.
+
링크 형식은 다음과 같습니다. `\https://URL` .

. 유지 관리 사용자 자격 증명을 사용하여 Unified Manager 웹 UI에 로그인합니다.



NOTE: 1시간 이내에 웹 UI에 로그인을 3회 연속 실패하면 시스템에서 잠기게 되며, 시스템 관리자에게 문의해야 합니다.  이는 로컬 사용자에게만 적용됩니다.
