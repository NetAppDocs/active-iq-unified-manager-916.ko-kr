= 세션 비활성 시간 초과 설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager에 대한 비활성 시간 초과 값을 지정하여 일정 시간 동안 비활성 상태가 지속되면 세션이 자동으로 종료되도록 할 수 있습니다.  기본적으로 제한 시간은 4,320분(72시간)으로 설정됩니다.

.시작하기 전에
애플리케이션 관리자 역할이 있어야 합니다.

이 설정은 로그인한 모든 사용자 세션에 영향을 미칩니다.

[NOTE]
====
SAML(Security Assertion Markup Language) 인증을 활성화한 경우 이 옵션을 사용할 수 없습니다.

====
.단계
. 왼쪽 탐색 창에서 *일반* > *기능 설정*을 클릭합니다.
. *기능 설정* 페이지에서 다음 옵션 중 하나를 선택하여 비활성 시간 초과를 지정합니다.
+
[cols="2*"]
|===
| 만약 당신이 원한다면... | 그러면 이렇게 하세요... 


 a| 
세션이 자동으로 닫히지 않도록 시간 초과를 설정하지 마십시오.
 a| 
*비활성 시간 초과* 패널에서 슬라이더 버튼을 왼쪽(끄기)으로 옮기고 *적용*을 클릭합니다.



 a| 
시간 초과 값으로 특정 분 수를 설정합니다.
 a| 
*비활성 시간 초과* 패널에서 슬라이더 버튼을 오른쪽(켜짐)으로 옮기고, 비활성 시간 초과 값을 분 단위로 지정한 다음 *적용*을 클릭합니다.

|===
