= Active IQ Unified Manager 구성
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Active IQ Unified Manager (이전 OnCommand Unified Manager)를 설치한 후 웹 UI에 액세스하려면 초기 설정(첫 번째 경험 마법사라고도 함)을 완료해야 합니다.  그런 다음 클러스터 추가, 원격 인증 구성, 사용자 추가, 알림 추가 등의 추가 구성 작업을 수행할 수 있습니다.

이 매뉴얼에 설명된 일부 절차는 Unified Manager 인스턴스의 초기 설정을 완료하는 데 필요합니다.  다른 절차로는 새 인스턴스에 설정하는 데 도움이 되는 권장 구성 설정이나 ONTAP 시스템의 정기 모니터링을 시작하기 전에 알아두면 좋은 내용이 있습니다.
