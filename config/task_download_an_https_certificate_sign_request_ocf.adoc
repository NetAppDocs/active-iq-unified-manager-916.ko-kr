= HTTPS 인증서 서명 요청 다운로드
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
현재 HTTPS 보안 인증서에 대한 인증서 서명 요청서를 다운로드하여 인증 기관에 파일을 제공하여 서명을 받을 수 있습니다.  CA 서명 인증서는 중간자 공격을 방지하는 데 도움이 되며 자체 서명 인증서보다 더 나은 보안 기능을 제공합니다.

.시작하기 전에
애플리케이션 관리자 역할이 있어야 합니다.

.단계
. 왼쪽 탐색 창에서 *일반* > *HTTPS 인증서*를 클릭합니다.
. *HTTPS 인증서 서명 요청 다운로드*를 클릭합니다.
. 저장하다 `<hostname>.csr` 파일.


인증 기관에 파일을 제공하여 서명을 받은 다음, 서명된 인증서를 설치할 수 있습니다.
