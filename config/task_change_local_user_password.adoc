= 로컬 사용자 비밀번호 변경
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
잠재적인 보안 위험을 방지하기 위해 로컬 사용자 로그인 비밀번호를 변경할 수 있습니다.

.시작하기 전에
로컬 사용자로 로그인해야 합니다.

유지 관리 사용자와 원격 사용자의 비밀번호는 이 단계를 통해 변경할 수 없습니다.  원격 사용자 비밀번호를 변경하려면 비밀번호 관리자에게 문의하세요.  유지 관리 사용자 비밀번호를 변경하려면 다음을 참조하세요.link:task_use_maintenance_console.html["유지 관리 콘솔 사용"] .

.단계
. Unified Manager에 로그인하세요.
. 상단 메뉴바에서 사용자 아이콘을 클릭한 다음 *비밀번호 변경*을 클릭합니다.
+
원격 사용자의 경우 *비밀번호 변경* 옵션이 표시되지 않습니다.

. 비밀번호 변경 대화 상자에서 현재 비밀번호와 새 비밀번호를 입력합니다.
. *저장*을 클릭하세요.


Unified Manager가 고가용성 구성으로 구성된 경우 설정의 두 번째 노드에서 비밀번호를 변경해야 합니다.  두 인스턴스 모두 동일한 비밀번호를 가져야 합니다.
