= 사용자 정의 umadmin 사용자 및 유지 관리 그룹 제거
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager를 설치하기 전에 사용자 정의 홈 디렉토리를 만들어 고유한 umadmin 사용자 및 유지 관리 계정을 정의한 경우 Unified Manager를 제거한 후 이러한 항목을 제거해야 합니다.

표준 Unified Manager 제거에서는 사용자 정의 umadmin 사용자와 유지 관리 계정이 제거되지 않습니다.  이러한 항목은 수동으로 삭제해야 합니다.

.단계
. Red Hat Enterprise Linux 서버에 루트 사용자로 로그인합니다.
. umadmin 사용자를 삭제합니다.
+
`userdel umadmin`

. 유지 관리 그룹을 삭제합니다.
+
`groupdel maintenance`
