= 클러스터 재발견
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
클러스터 설정 페이지에서 클러스터를 수동으로 다시 검색하여 클러스터의 상태, 모니터링 상태 및 성능 상태에 대한 최신 정보를 얻을 수 있습니다.

클러스터를 업데이트하려는 경우(예: 공간이 부족하여 집계 크기를 늘리는 경우) 클러스터를 수동으로 다시 검색할 수 있으며 Unified Manager에서 변경 사항을 검색하도록 할 수 있습니다.

Unified Manager가 OnCommand Workflow Automation (WFA)과 페어링되면 페어링을 통해 WFA에서 캐시된 데이터가 다시 수집됩니다.

.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *클러스터 설정*을 클릭합니다.
. *클러스터 설정* 페이지에서 *재검색*을 클릭합니다.
+
Unified Manager는 선택된 클러스터를 다시 검색하고 최신 상태 및 성능 상태를 표시합니다.
