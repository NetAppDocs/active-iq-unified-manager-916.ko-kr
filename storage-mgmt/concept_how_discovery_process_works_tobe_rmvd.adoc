= 발견 프로세스가 작동하는 방식
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager에 클러스터를 추가하면 서버에서 클러스터 개체를 검색하여 데이터베이스에 추가합니다.  검색 프로세스가 작동하는 방식을 이해하면 조직의 클러스터와 해당 개체를 관리하는 데 도움이 됩니다.

기본 모니터링 간격은 15분입니다. Unified Manager 서버에 클러스터를 추가한 경우 Unified Manager UI에 클러스터 세부 정보가 표시되는 데 15분이 걸립니다.
