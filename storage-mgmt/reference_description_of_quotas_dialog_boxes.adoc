= 할당량 대화 상자 설명
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Health: All Storage VMs 보기의 User and Group Quotas 탭에서 적절한 옵션을 사용하여 할당량 관련 문제가 발생할 때 전송되는 이메일 알림의 형식을 구성하고 사용자 할당량에 따라 이메일 주소를 지정하는 규칙을 구성할 수 있습니다.
