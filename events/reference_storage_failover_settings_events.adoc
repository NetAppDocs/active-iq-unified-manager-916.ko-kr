= 스토리지 장애 조치 설정 이벤트

:icons: font
:imagesdir: ../media/


[role="lead"]
저장소 장애 조치(SFO) 설정 이벤트는 저장소 장애 조치가 비활성화되었는지 또는 구성되지 않았는지에 대한 정보를 제공하여 잠재적인 문제를 모니터링할 수 있도록 합니다.  이벤트는 영향 영역별로 그룹화되며 이벤트 및 트랩 이름, 영향 수준, 소스 유형, 심각도가 포함됩니다.
