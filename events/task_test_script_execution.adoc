= 테스트 스크립트 실행
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
저장소 개체에 대한 알림이 생성되면 스크립트가 올바르게 실행되는지 확인할 수 있습니다.

.시작하기 전에
* 애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.
* 지원되는 파일 형식의 스크립트를 Unified Manager에 업로드해야 합니다.


.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *스크립트*를 클릭합니다.
. *스크립트* 페이지에서 테스트 스크립트를 추가합니다.
. 왼쪽 탐색 창에서 *저장소 관리* > *알림 설정*을 클릭합니다.
. *알림 설정* 페이지에서 다음 작업 중 하나를 수행합니다.
+
|===
| 에게... | 이렇게 하세요... 


 a| 
알림 추가
 a| 
.. *추가*를 클릭하세요.
.. 작업 섹션에서 알림을 테스트 스크립트와 연결합니다.




 a| 
알림 편집
 a| 
.. 알림을 선택한 다음 *편집*을 클릭합니다.
.. 작업 섹션에서 알림을 테스트 스크립트와 연결합니다.


|===
. *저장*을 클릭하세요.
. *알림 설정* 페이지에서 추가하거나 수정한 알림을 선택한 다음 *테스트*를 클릭합니다.
+
스크립트는 "-test" 인수로 실행되고, 알림 경고는 경고가 생성될 때 지정된 이메일 주소로 전송됩니다.
