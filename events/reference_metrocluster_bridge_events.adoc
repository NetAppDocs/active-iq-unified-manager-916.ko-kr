= MetroCluster 브리지 이벤트
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
MetroCluster Bridge 이벤트는 FC 구성을 통한 MetroCluster 에서 잠재적인 문제를 모니터링할 수 있도록 브리지 상태에 대한 정보를 제공합니다.  이벤트는 영향 영역별로 그룹화되며 이벤트 및 트랩 이름, 영향 수준, 소스 유형, 심각도가 포함됩니다.
