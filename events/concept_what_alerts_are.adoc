= 어떤 알림이 있나요?
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이벤트는 지속적으로 발생하지만 Unified Manager는 이벤트가 지정된 필터 기준을 충족할 때만 알림을 생성합니다.  알림을 생성할 이벤트를 선택할 수 있습니다. 예를 들어, 공간 임계값을 초과하거나 개체가 오프라인이 되는 경우입니다.  알림이 발생할 때 실행되는 스크립트와 알림을 연결할 수도 있습니다.

필터 기준에는 개체 클래스, 이름 또는 이벤트 심각도가 포함됩니다.
