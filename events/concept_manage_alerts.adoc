= 알림 관리
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
특정 이벤트나 특정 심각도 유형의 이벤트가 발생하면 자동으로 알림을 보내도록 알림을 구성할 수 있습니다.  알림이 발생할 때 실행되는 스크립트와 알림을 연결할 수도 있습니다.
