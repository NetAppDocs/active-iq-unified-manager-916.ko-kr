= 스크립트 추가 대화 상자
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
스크립트 추가 대화 상자를 사용하면 Unified Manager에 스크립트를 추가할 수 있습니다.  스크립트를 사용하여 알림을 구성하여 스토리지 개체에 대해 생성된 이벤트를 자동으로 해결할 수 있습니다.

애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

* *스크립트 파일 선택*
+
알림에 대한 스크립트를 선택할 수 있습니다.

* *설명*
+
스크립트에 대한 설명을 지정할 수 있습니다.
