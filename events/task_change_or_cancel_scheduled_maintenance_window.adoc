= 예약된 유지 관리 기간을 변경하거나 취소합니다.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager 유지 관리 기간이 미래에 발생하도록 구성한 경우 시작 및 종료 시간을 변경하거나 유지 관리 기간이 발생하지 않도록 취소할 수 있습니다.

.시작하기 전에
애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

예약된 유지 관리 기간 종료 시간 전에 클러스터 유지 관리를 완료하고 클러스터에서 이벤트와 알림을 다시 받으려는 경우 현재 실행 중인 유지 관리 기간을 취소하는 것이 유용합니다.

.단계
. 왼쪽 탐색 창에서 *저장소 관리* > *클러스터 설정*을 클릭합니다.
. 클러스터의 *유지 관리 모드* 열에서:
+
|===
| 만약 당신이 원한다면... | 이 단계를 수행하세요... 


 a| 
예약된 유지 관리 기간의 시간 프레임 변경
 a| 
.. 슬라이더 버튼 옆에 있는 "예약됨"이라는 텍스트를 클릭합니다.
.. 시작 및/또는 종료 날짜와 시간을 변경하고 *적용*을 클릭합니다.




 a| 
활성 유지 관리 기간의 길이를 연장합니다.
 a| 
.. 슬라이더 버튼 옆에 있는 "활성"이라는 텍스트를 클릭하세요.
.. 종료 날짜와 시간을 변경하고 *적용*을 클릭하세요.




 a| 
예약된 유지 관리 기간 취소
 a| 
슬라이더 버튼을 선택하여 왼쪽으로 이동하세요.



 a| 
활성 유지 관리 창 취소
 a| 
슬라이더 버튼을 선택하여 왼쪽으로 이동하세요.

|===
