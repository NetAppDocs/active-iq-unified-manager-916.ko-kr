= 성능 이벤트 해결
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
제안된 작업을 사용하여 성능 이벤트를 직접 해결해 볼 수 있습니다.  처음 세 가지 제안은 항상 표시되고, 네 번째 제안에 따른 작업은 표시되는 이벤트 유형에 따라 달라집니다.

*도움이 필요합니다* 링크는 특정 작업을 수행하기 위한 지침을 포함하여 제안된 각 작업에 대한 추가 정보를 제공합니다.  일부 작업에는 Unified Manager, ONTAP System Manager, OnCommand Workflow Automation, ONTAP CLI 명령 또는 이러한 도구의 조합을 사용하는 것이 포함될 수 있습니다.
