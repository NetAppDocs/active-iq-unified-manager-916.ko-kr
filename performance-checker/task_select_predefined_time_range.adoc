= 미리 정의된 시간 범위를 선택하세요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
미리 정의된 시간 범위를 선택하면 클러스터 개체 성능 데이터를 볼 때 데이터 출력을 사용자 지정하고 집중할 수 있는 빠르고 효율적인 방법입니다.  미리 정의된 시간 범위를 선택하면 최대 13개월 동안의 데이터를 사용할 수 있습니다.

.단계
. *성능 탐색기* 페이지의 오른쪽 상단에서 *시간 범위*를 클릭합니다.
. *시간 범위 선택* 패널의 오른쪽에서 미리 정의된 시간 범위를 선택합니다.
. *범위 적용*을 클릭합니다.
