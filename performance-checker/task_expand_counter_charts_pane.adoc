= 카운터 차트 창 확장
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
카운터 차트 창을 확장하면 차트를 더 크고 읽기 쉽게 볼 수 있습니다.

비교 개체와 카운터의 시간 범위를 정의한 후에는 더 큰 카운터 차트 창을 볼 수 있습니다.  성능 탐색기 창 가운데에 있는 **<* 버튼을 사용하여 창을 확장합니다.

.단계
. *카운터 차트* 창을 확장하거나 축소합니다.


|===
| 에게... | 이렇게 하세요... 


 a| 
페이지 너비에 맞게 카운터 차트 창을 확장합니다.
 a| 
*<* 버튼을 클릭하세요



 a| 
카운터 차트 창을 페이지의 오른쪽 절반으로 줄입니다.
 a| 
*>* 버튼을 클릭하세요

|===
