= 사용자 정의 성능 임계값에서 이벤트 분석
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용자 정의 임계값에서 생성된 이벤트는 특정 스토리지 개체(예: 집계 또는 볼륨)에 대한 성능 카운터가 정책에서 정의한 임계값을 넘어섰음을 나타냅니다.  이는 클러스터 개체에 성능 문제가 발생하고 있음을 나타냅니다.

이벤트 세부 정보 페이지를 사용하여 성능 이벤트를 분석하고 필요한 경우 시정 조치를 취해 성능을 정상으로 되돌릴 수 있습니다.
